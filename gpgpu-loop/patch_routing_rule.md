# Patch Routing Rule

Patch classes include:

- `ARCHITECTURE_PATCH`
- `CONTRACT_PATCH`
- `RTL_PATCH`
- `MEMORY_PATCH`
- `INTERCONNECT_PATCH`
- `ATOMIC_SYNC_PATCH`
- `RUNTIME_ABI_PATCH`
- `COUNTER_SCHEMA_PATCH`
- `TEST_EVIDENCE_PATCH`
- `SIMULATOR_ARTIFACT_REMOVAL_PATCH`

Route by owner, not by symptom wording.
